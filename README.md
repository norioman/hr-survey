# 人事制度 支持率投票アプリ 使い方ガイド

## 概要

人事制度説明会で使用する、匿名の支持率投票アプリです。
査定対象者8名がそれぞれのPCから投票し、結果をスプレッドシートで集計します。

---

## システム構成

```
投票者のブラウザ（GitHub Pages）
　　↓ 投票データを送信
Google Apps Script（APIの窓口）
　　↓ 書き込み・読み込み
Google スプレッドシート（データ保存・集計）
```

---

## 初回セットアップ

### 1. スプレッドシートを作成する

1. Google スプレッドシートを新規作成
2. シート名を `votes_1` に変更（第1回投票用）
3. URLの `/d/` と `/edit` の間の文字列をメモする（スプレッドシートID）

例：
```
https://docs.google.com/spreadsheets/d/★ここがID★/edit
```

---

### 2. Google Apps Script をデプロイする

1. スプレッドシートのメニュー → **拡張機能** → **Apps Script**
2. 既存のコードを全削除し、`Code.gs` の内容を貼り付ける
3. `SPREADSHEET_ID` を手順1のIDに書き換えて保存（Ctrl+S）

```javascript
const SPREADSHEET_ID = "ここに貼り付ける";
```

4. 右上「デプロイ」→「新しいデプロイ」
5. 種類の選択 → ⚙️ → **ウェブアプリ**
6. 以下のように設定してデプロイ

| 項目 | 設定値 |
|------|--------|
| 次のユーザーとして実行 | 自分 |
| アクセスできるユーザー | 全員 |

7. 初回のみ権限承認が求められる → **許可**
8. 表示された**ウェブアプリのURL**をコピーしてメモ

> ⚠️ コードを修正した場合は「デプロイを管理」→ 編集（鉛筆アイコン）→「新しいバージョン」で更新。URLは変わりません。

---

### 3. index.html を編集する

`index.html` の以下の2箇所を書き換える。

```javascript
const GAS_URL = "★ここに手順2のURLを貼り付ける★";
const CURRENT_SHEET = "votes_1"; // 現在の投票回のシート名
```

---

### 4. GitHub Pages に公開する

1. GitHubで新しいリポジトリを作成（Public）
2. `index.html` をアップロード
3. **Settings** → **Pages** → Branch: `main` / `/(root)` → **Save**
4. 数分後に公開URLが発行される

例：
```
https://（ユーザー名）.github.io/（リポジトリ名）/
```

5. このURLを8名に共有する

---

## 投票の使い方（投票者向け）

### 投票手順

1. 共有されたURLをブラウザで開く
2. **「📝 投票」タブ**が表示される
3. 各項目の「▼ 説明」を押すと制度の詳細が確認できる
4. 各項目に **👍 賛成 / 🤔 中立 / 👎 反対** のいずれかを選ぶ
5. 全項目への投票が終わったら下部の「**投票を送信**」を押す
6. 「✓ 送信済み」バッジが表示されれば完了

### 投票を変更したい場合

- 選択肢を押し直してから「**投票を送信**」を再度押すだけ
- 同じPCからの再送信はスプレッドシート上で**上書き**される（行は増えない）
- 前回の選択はブラウザに保存されているため、再度開いても復元される

> ⚠️ 別のPC・別のブラウザから送信すると別人扱いになります。必ず自分のPCから投票してください。

## 結果の確認（管理者向け）

### 結果を表示する

1. **「📊 結果」タブ**を開く
2. 「対象回」で確認したい回を選択（例：第1回）
3. 「**表示**」を押す
4. 総合スコアと項目別の支持率バーが表示される

### スコアの見方

総合スコアは全22項目の加重平均です。

```
スコア = (賛成票 × 1点 + 中立票 × 0.5点) ÷ 総投票数 × 100
```

| スコア | 色 | 意味 |
|--------|-----|------|
| 60点以上 | 🟢 緑 | 支持が高い |
| 40〜59点 | 🟡 黄 | 賛否が分かれている |
| 39点以下 | 🔴 赤 | 反対・懸念が多い |

### 回をまたいで比較する

1. 「対象回」に今回、「比較回」に前回を選択
2. 「**表示**」を押す
3. 総合スコアカードに差分（例: `+8pt`）が表示される
4. 項目ごとにも前回との差分バッジが表示される

---

## 次の回の投票を始める

1. `Code.gs` の `resetVotes()` 関数内のシート名を次の回に変更して手動実行

```javascript
function resetVotes() {
  const sheetName = "votes_2"; // ← 次の回のシート名に変更
  ...
}
```

2. `index.html` の `CURRENT_SHEET` を書き換えてGitHubに再アップロード

```javascript
const CURRENT_SHEET = "votes_2";
```

3. 結果タブには第1回・第2回が両方表示され、比較できる

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| 送信しても「⚠ 送信に失敗しました」 | GAS_URLが未設定 or GASが未デプロイ | URLを確認・再デプロイ |
| 投票件数が8件を超えた | 同じ人が複数PCから送信 | スプレッドシートで重複行を手動削除し、resetVotes()で再スタート |
| 結果タブにシートが表示されない | シート名が`votes_`で始まっていない | スプレッドシートのシート名を確認 |
| 前回の選択が消えた | 別のブラウザ・シークレットモードを使用 | 通常のブラウザで開き直す |
